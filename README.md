# GreenLeaf Plant Shop

A plant shop website with 7 pages, wishlist, filter, and contact form.

## Pages
- Home
- Shop
- About
- Contact
- Plant Care
- FAQ
- Wishlist

## Features
- Wishlist (save plants)
- Filter (Low Light / Pet Friendly)
- Contact form validation
- Newsletter signup

## Technologies
- HTML
- CSS
- Bootstrap
- JavaScript
- Font Awesome
- Git & GitHub

## How to Run
Open index.html in your browser.
